Css transform позволяет трансформировать элементы, в том числе в трехмерном пространстве.

transform: rotate() (угол)

Поворачивает элементы на заданное количество градусов, отрицательные значения от -1deg до -360deg поворачивают элемент против часовой стрелки, положительные — по часовой стрелке. Значение rotate(720deg) поворачивает элемент на два полных оборота.

rotate(360deg)
rotate(720deg)

transform: perspective();

В CSS3 была добавлена еще одна функция — perspective(), которая может быть использована для создания эффекта перспективы, глубины в сцене. Перемещая элемент по оси Z с установленной перспективой, можно наблюдать, как он приближается или отдаляется от зрителя.
