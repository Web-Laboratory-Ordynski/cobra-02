transform:rotate3d(x,y,z,угол)

.rotate3d {
transform: rotate3d(2, 1, 5, 50deg);
background-color: aqua;
position: absolute;
margin-left: 800px;
}

Функция вращает элемент по часовой стрелке относительно трех осей.
Элемент поворачивается под углом, задаваемым последним параметром
относительно вектора направления [x,y,z].
Отрицательные значения поворачивают элемент против часовой стрелки.

rotate3d(2, 1, 5, 50deg)

transform:rotateX(deg)

.rotateX {
transform: rotateX(180deg);
}

Функция задает поворот по часовой стрелке под заданным углом относительно оси X.
Функция rotateX(180deg) эквивалентна rotate3d(1,0,0,180deg).
transform: rotateX(30deg);
transform: rotateX(-135deg);
